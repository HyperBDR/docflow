# DocFlow Recorder

## 构建与安装

```bash
npm install
npm run build
```

在 Chrome 或 Edge 打开扩展管理页，启用开发者模式，选择“加载已解压的扩展”，加载 `extension/dist`。

## 使用

1. 在 DocFlow Web 首页点击“连接浏览器扩展”，取得六位配对码。
2. 在扩展中填写 API 地址和配对码。
3. 在扩展中选择一个演示并点击开始录制。当前网页会打开页面内设置弹窗，可选择 **HTML Cloning** / **Screenshot**，并为本次录制单独开启或关闭 AI。
4. 阅读三步录制教学并点击 `Get Started`。界面会自动根据浏览器语言使用中文或英文。
5. HTML Cloning 录制时移动鼠标会高亮候选元素；点击后等待短暂上传完成，再选择下一个元素。
6. 页面左下角控制条可拖动，支持 Pause / Resume、查看步骤数量、无热点手动截图和 Stop Recording。停止后编辑器会自动打开。

每次选择元素后，扩展只在上传截图、经过清洗的 DOM 快照、目标元素选择器和滚动位置时短暂锁定页面，服务端确认入库后立即允许继续选择。如果本次录制启用了 AI，API 会自动把步骤提交给后台 Worker，标题、说明和热点提示生成不会阻塞后续录制。手动截图会保存当前 HTML 与截图但不创建热点。控制条、模式弹窗和教学弹窗不会进入截图或 DOM 快照。HTML 模式会尝试通过扩展权限内联跨域样式表、字体和图片；访问受限的资源仍使用截图作为视觉回退。人工修改过的字段不会被后续 AI 结果覆盖。

更新扩展代码并重新执行 `npm run build` 后，需要回到扩展管理页点击“重新加载”，已打开的目标页面也建议刷新一次。

浏览器内部页面、扩展商店及部分跨域 iframe 受浏览器安全策略限制，不能录制；这些步骤可在编辑器中手动上传截图。
