# DocFlow Recorder

## 构建与安装

```bash
npm install
npm run build
```

在 Chrome 或 Edge 打开扩展管理页，启用开发者模式，选择“加载已解压的扩展”，加载 `extension/dist`。

## 使用

1. 在 DocFlow Web 首页点击“连接浏览器扩展”，取得六位配对码。
2. 在扩展中填写 API 地址和配对码。
3. 选择一个已创建的演示和录制模式。默认的 **HTML Cloning** 会克隆 HTML、CSS 和可访问的图片资源；Screenshot 模式只保存静态截图。
4. HTML Cloning 录制时移动鼠标会高亮候选元素；点击后等待页面底部的 `Capturing, please wait…` 消失，再选择下一个元素。
5. 扩展弹窗和图标会显示已录制步数，可随时 Pause / Resume；点击 Stop Recording 后编辑器会自动打开。

每次选择元素后，扩展只在上传截图、经过清洗的 DOM 快照、目标元素选择器和滚动位置时短暂锁定页面，服务端确认入库后立即允许继续选择。如果启用了 AI，API 会自动把本步骤提交给后台 Worker，标题、说明和热点提示生成不会阻塞后续录制。HTML 模式会尝试通过扩展权限内联跨域样式表、字体和图片；访问受限的资源仍使用截图作为视觉回退。人工修改过的字段不会被后续 AI 结果覆盖。

更新扩展代码并重新执行 `npm run build` 后，需要回到扩展管理页点击“重新加载”，已打开的目标页面也建议刷新一次。

浏览器内部页面、扩展商店及部分跨域 iframe 受浏览器安全策略限制，不能录制；这些步骤可在编辑器中手动上传截图。
